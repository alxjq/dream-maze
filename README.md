# IMPOSSIBLE BACKROOMS

A psychological 3D raycasting maze game built with Ebiten in Go.

The maze grows.
The fog thickens.
The walls deceive.
Escape becomes unlikely.

After all these years... you finally wake up, Terry.

---

## Features

- Procedurally generated maze (DFS + chaos loops)
- Increasing maze size every level
- Dynamic fog system (visibility decreases per level)
- Level-based psychological messages
- Unique wall colors per stage
- Final brain awakening screen
- Restartable ending (Press R)

---

## Controls

| Key | Action |
|------|--------|
| W | Move Forward |
| S | Move Backward |
| A | Rotate Left |
| D | Rotate Right |
| R | Restart (Ending Screen) |

---

## Levels

Level 1  
Escape probability extremely low (crying sounds)

Level 2  
Brain death probability eliminated (sobbing)

Level 3  
Cognitive functions improving (joy screams)

Level 4  
...

Golden walls. Blue exit. Silence.
